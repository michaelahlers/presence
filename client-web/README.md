# Michael Ahlers Presence Web Client

## Development

### Prerequisites

#### OS X

1. Follow installation instructions for [Homebrew](http://brew.sh/).
1. `$ brew install node`
1. `$ npm install`
  - Whenever dependencies are changed.

### Building

1. `$ npm run watch`
  - Will watch source trees for changes.
